Troubleshooting

Only Google Chrome is supported by this launcher.

Please wait for the initialization.

Please install pcman-chrome before connecting.

If you use a customized pcman-chrome app with a different ID from it:

Please goto "chrome://extensions/" and check 'Developer mode' to get your app's id.
Replace the field as your app's id and reload bbs pages.

Please manually start the server of the pcman-chrome app.
When 'Run in the background' is set, the server starts automatically and this step is skipped.
